Welcome to Codebuddies!

Please find some useful commands below to help you get started on Django development

Application urls

  • /api/v1/resources/ - browse the API for resources
  • /admin/ - login using superuser credentials
  • /auth/obtain_token/admin/ - get a token
  • /auth/users/ - create a user

docker-compose commands

  • docker-compose run --rm app ./manage.py createsuperuser - Command to create a superuser
  • docker-compose run --rm app pytest - Command to run tests
  • docker-compose run --rm app ./manage.py makemigrations - Command to make migrations (necessary if you change the models)
  • docker-compose run --rm app ./manage.py migrate - Command to migrate the migrations (necessary if you ran the previous makemigrations step)
